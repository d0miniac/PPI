O que é JavaScript?

O JavaScript foi criado para adicionar interatividade às páginas da web.
Foi desenvolvido pela Netscape em 1995 como uma linguagem leve para o navegador.
Diferente de outras linguagens, o JavaScript pode ser executado tanto no navegador quanto no servidor
(com Node.js). No navegador, ele opera em uma sandbox, um ambiente seguro que restringe seu acesso ao sistema.
Alguns objetos globais essenciais do JavaScript são document (para manipular elementos HTML),
console (para depuração) e alert (para exibir mensagens na tela).
